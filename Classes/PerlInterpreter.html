PerlInterpreter

public final class PerlInterpreter

A safe Perl Interpreter.

Instances of this class are useful for embedding Perl Interpreter into your Swift program. For other cases this implementation is mostly abandoned for performance reasons and should be rethinked.

  • withUnsafeInterpreterPointer(_:)

    Invokes the given closure on the unsafe pointer to the Perl Interpreter.

    The withUnsafeInterpreterPointer(_:) method ensures that the Interpreter’s lifetime extends through the execution of body.

    Parameter

    Parameter body: A closure that takes unsafe pointer to the Perl Interpreter as its sole argument. If the closure has a return value, it is used as the return value of the withUnsafeInterpreterPointer(_:) method.

    Returns

    The return value of the body closure, if any.

    Calls Perl subroutine by its name.
